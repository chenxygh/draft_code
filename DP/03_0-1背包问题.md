# 03_0-1背包问题

---

注意细节， 必须对于临界条件的判断， 以及算法上的优化， 要熟练

解题思路:

1. 最优子结构

做出选择， F(n - 1, c - w[n]) + v[n], F(n - 1, c)

2. 状态转移

F(n) = max (c >= w[n]? F(n - 1, c - w[n]) + v[n], F(n - 1, c));

3. 边界

n <= 0 || c <= 0     0
